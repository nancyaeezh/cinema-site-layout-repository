# cinema-site-layout-repository
该项目围绕电影类站点在页面布局资料仓体系与导航路径方面的公开资料进行整理，通过归档栏目命名逻辑、页面分区设计与关键词结构布局，形成系统化研究样本。仓库不包含影视播放或下载内容，仅对公开页面元素进行记录说明。内容重点关注首页与内页之间的层级关系与组织模式，用于页面结构研究与对比分析。所有资料来源于公开渠道，仅供信息整理与结构分析参考。
